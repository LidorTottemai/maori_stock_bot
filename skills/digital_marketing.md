## Google Ads — מה אתה יודע

### Search Campaigns
- מבנה חשבון נכון: Campaign → Ad Group → Keywords → Ads
- Match types: Exact [keyword], Phrase "keyword", Broad ~keyword
- Quality Score: CTR + Ad Relevance + Landing Page = המחיר שתשלם
- Smart Bidding: Target CPA, Target ROAS, Maximize Conversions
- Extensions: Sitelinks, Callouts, Structured Snippets, Call, Location

### Performance Max
- מתי להשתמש: כשיש מספיק קונברז'נים (50+/חודש)
- מה לספק: Assets מגוונים, Audience signals חכמים, Conversion tracking מדויק
- אזהרה: קופסה שחורה — צריך לסמוך על הדאטה

### Shopping / Retail
- Product Feed איכותי = חצי מהעבודה
- Titles עם Keywords ↑ CTR
- Smart Shopping vs. Standard — תלוי בשלב

## Meta Ads — מה אתה יודע

### מבנה קמפיין
```
Campaign (מטרה: Conversions / Traffic / Awareness)
  └── Ad Set (קהל + תקציב + מיקום)
        └── Ad (Creative: Image / Video / Carousel / Story)
```

### קהלים
- Core Audience: דמוגרפי + תחומי עניין
- Custom Audience: רשימת לקוחות, Website visitors, App users, Video viewers
- Lookalike: 1%-3% מ-Custom Audience איכותי

### Creative Strategy
- **Hook:** 3 שניות ראשונות — קונה או מפסיד
- **Body:** הוכחה חברתית + יתרונות
- **CTA:** ברור, חד, ישיר
- **Testing:** 3-5 creatives לאד-סט, Kill מה שלא מביא תוצאות תוך 48-72 שעות

### iOS 14+ עולם חדש
- Aggregated Event Measurement — עד 8 Events
- Conversion API (CAPI) = חובה
- Attribution window: 7-day click / 1-day view

## SEO — מה אתה יודע

### Pillars
```
Technical SEO        On-Page SEO          Off-Page SEO
──────────────       ──────────────       ──────────────
Site speed           Keyword research     Backlinks
Mobile-first         Title tags           Domain Authority
Core Web Vitals      Meta descriptions    Brand mentions
Crawlability         Internal linking     Social signals
Schema markup        Content quality      Digital PR
```

### Keyword Research Process
1. Seed keywords מהעסק
2. כלים: Ahrefs, SEMrush, Google Keyword Planner
3. Cluster לפי Intent: Informational / Navigational / Commercial / Transactional
4. Prioritize: Volume + Difficulty + Business Value
5. Build Content Plan לפי clusters

### Technical Checklist
- [ ] Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1
- [ ] HTTPS + Canonical tags
- [ ] XML Sitemap + Robots.txt
- [ ] Schema markup (Product, FAQ, Review, LocalBusiness)
- [ ] Mobile responsive
- [ ] No broken links / redirect chains
