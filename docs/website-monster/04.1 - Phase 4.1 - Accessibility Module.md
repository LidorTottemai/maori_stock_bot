# ♿ Phase 4.1 — מודול נגישות (חובה חוקית)

> **חוק:** תקן ישראלי 5568, חוק שוויון זכויות לאנשים עם מוגבלות (1998)
> **תקן:** WCAG 2.0 Level AA
> **קנס:** עד ₪50,000 לאתר ללא נגישות, תביעות פרטיות ללא הוכחת נזק
> **מקור הרכיבים:** `@tottemai/ui` — מותקן בכל אתר, לא לוקאלי

---

## מה מחייב החוק?

```
1. נגישות WCAG 2.0 AA — keyboard, contrast, screen reader, alt text
2. כפתור נגישות (AccessibilityWidget) — convention חזק בשוק הישראלי
3. הצהרת נגישות (/accessibility-statement) — חובה, כולל תאריך בדיקה + רכז נגישות
4. עדכון שנתי — תאריך הבדיקה מוזרק אוטומטית בbuild
```

## רכיבים מ-@tottemai/ui

```ts
import { SkipLink, AccessibilityWidget, AccessibilityStatement } from "@tottemai/ui"
```

ועוד CSS חובה:
```css
/* globals.css */
@import "@tottemai/ui/a11y.css";
```

| רכיב | תיאור | מסמך |
|------|-------|-------|
| `useA11yStore` | Zustand store + persist — מקור האמת לכל הגדרות הנגישות | [[component-library/a11y/useA11yStore]] |
| `AccessibilityWidget` | כפתור צף + פאנל 10 כלים — כותב ל-store | [[component-library/a11y/AccessibilityWidget]] |
| `A11yProvider` | מאזין לstore → מפעיל CSS vars + filter + classes על `<html>` | [[component-library/a11y/A11yProvider]] |
| `SkipLink` | דלג לתוכן — sr-only → focus:visible | [[component-library/a11y/SkipLink]] |
| `AccessibilityStatement` | דף הצהרת נגישות מלאה | [[component-library/a11y/AccessibilityStatement]] |

## שילוב ב-layout.tsx

```tsx
// app/[locale]/layout.tsx
import { A11yProvider, SkipLink, AccessibilityWidget } from "@tottemai/ui"
import "@tottemai/ui/a11y.css"

export default function RootLayout({
  children,
  params: { locale },
}: {
  children: React.ReactNode
  params: { locale: string }
}) {
  return (
    <html lang={locale} dir={locale === "he" ? "rtl" : "ltr"}>
      <body>
        {/* A11yProvider — לא מרנדר HTML, רק מאזין לZustand ומפעיל CSS */}
        <A11yProvider />

        <SkipLink locale={locale as "he" | "en"} />   {/* ראשון הנראה בDOM */}

        <Navbar />
        <main id="main-content">
          {children}
        </main>
        <Footer />

        <AccessibilityWidget                           {/* אחרון בDOM */}
          locale={locale as "he" | "en"}
          businessName={process.env.NEXT_PUBLIC_BUSINESS_NAME!}
          phone={process.env.NEXT_PUBLIC_PHONE!}
          email={process.env.NEXT_PUBLIC_EMAIL}
          statementUrl={`/${locale}/accessibility-statement`}
        />
      </body>
    </html>
  )
}
```

**Flow:** `AccessibilityWidget` כותב ל-`useA11yStore` → `A11yProvider` מגיב → CSS variables + filter + classes → העיצוב כולו מגיב בצורה נקייה, ללא `!important` overrides.

## WCAG 2.1 AA — כללים נוספים שנדחפים ל-CLAUDE.md

```
ACCESSIBILITY (WCAG 2.1 AA — חובה בכל אתר):

□ כל <img> חייב alt="" (דקורטיבי) או alt="תיאור"
□ כל interactive element — aria-label אם אין טקסט גלוי
□ Color contrast: טקסט רגיל ≥ 4.5:1, UI ≥ 3:1
□ Font sizes: rem בלבד. body minimum 1rem
□ Forms: <label> לכל <input>, error ב-aria-describedby
□ Modals: focus trap + Escape לסגירה + aria-modal="true"
□ Keyboard: Tab/Shift+Tab, Enter, Space, חצים בתפריטים
□ Focus outline: 2px solid var(--color-primary), offset: 2px — אסור display:none על :focus
□ prefers-reduced-motion: כל אנימציה עטופה (useReducedMotion מ-@tottemai/ui)
□ id="main-content" על <main> — חובה לSkipLink
```

## זיהוי אוטומטי ב-dig() (Phase 4)

```python
# site_archaeology.py — מחפש סימנים לwidget קיים
WIDGET_SELECTORS = [
    ".equalweb-btn", "#INDmenu-btn", ".userway-btn",
    "[data-userway]", ".adally-btn", "#nagich", ".ally-btn",
    "[id*='accessibility']", "[class*='accessibility-widget']",
]

async def _detect_accessibility_widget(page) -> bool:
    for sel in WIDGET_SELECTORS:
        if await page.query_selector(sel):
            return True
    return False
```
