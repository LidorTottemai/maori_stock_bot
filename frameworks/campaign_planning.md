# Framework: תכנון קמפיין

> ממבריף ועד ניתוח תוצאות — כל מה שצריך לכל קמפיין.

---

## שלב 1: הגדרת מטרה

### שאלות לפני הכל
1. **מה המטרה העסקית** מאחורי הקמפיין?
2. **מה ה-KPI שיוכיח הצלחה** — מספר אחד, לא חמישה?
3. **מה ה-Timeline?**
4. **מה התקציב?**

### Matrix: מטרה → ערוץ → מדד

| מטרה | ערוצים מומלצים | KPI ראשי |
|------|---------------|----------|
| Brand Awareness | Meta/TikTok Awareness, YouTube, PR | Reach, CPM, Brand Recall |
| Traffic | SEO, Google Search, Meta Traffic | CPC, CTR, Sessions |
| Lead Generation | Meta/LinkedIn Lead Gen, Google Search | CPL, Lead Quality |
| Sales | Google Shopping, Meta Conversions, Email | ROAS, CPA, Revenue |
| Retention | Email, Push, WhatsApp | Open Rate, Repeat Purchase |
| App Installs | Meta App, Google UAC, TikTok | CPI, D7/D30 Retention |

## שלב 2: הגדרת קהל

### Targeting Matrix

```
PRIMARY AUDIENCE
├── Who: [מי הם?]

├── Pain: [מה הכאב שלהם?]
└── Trigger: [מה יגרום להם לפעול?]

├── Who: [מי הם?]
├── Difference from Primary: [מה שונה?]

EXCLUSIONS
└── מי לא רוצים בקהל? (מתחרים, לקוחות קיימים, רשימות suppress)
```

### Creative Hypothesis
> אם נציג [קהל] עם [מסר] דרך [פורמט], הם יבצעו [פעולה] כי [סיבה].

|---------|------|--------|-----|-----|-------|

### Rules of Thumb

**Video:**
- 0-3 שניות: Hook (שאלה / claim / visual מפתיע)
- 3-15 שניות: Problem Agitation
- 15-45 שניות: Solution + Proof
- 45-60 שניות: CTA + Urgency

**Static:**
- Headline: ייחוד + תועלת
- Visual: מושך עין, רלוונטי לקהל
- Body: Proof points
- CTA: ברור, אחד

## שלב 5: Landing Page Checklist

- [ ] **Above the fold:** Headline ברור + Visual חזק + CTA נראה
- [ ] **Headline:** מייצג את הpromise מהמודעה (Message Match)
- [ ] **Trust Signals:** Reviews, Logos, Numbers, Certifications
- [ ] **Social Proof:** Testimonials עם שם + תמונה
- [ ] **CTA:** בולט, אחד, ברור מה קורה אחרי הלחיצה
- [ ] **Mobile Optimized:** 60-70% מהtraffic = mobile
- [ ] **Page Speed:** < 3 שניות (PageSpeed Insights)
- [ ] **Pixels:** Facebook Pixel + GA4 + Conversion Event
- [ ] **Heatmap:** Hotjar / Microsoft Clarity מותקן

## שלב 8: Optimization Protocol

### שבועי
```
בדוק:
☐ Spend vs. Budget — האם מתפרק נכון?
☐ Top performers — Scale ב-20-30%
☐ Underperformers — Kill אם CPA > Target × 1.5
☐ Creative freshness — חריגה מ-Frequency 3+?
☐ Landing Page CVR — ירידה? בדוק שינויים
```

### חודשי
```
בדוק:
☐ Unit Economics — CAC, LTV, ROAS trend
☐ Audience Saturation — הגדל קהלים / Lookalikes
☐ Competitor check — מה מריצים?
☐ Creative Refresh — בנה Batch חדש
☐ Channel Mix — האם ניצלנו הזדמנויות?
```
