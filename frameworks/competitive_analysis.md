# Framework: ניתוח מתחרים

> ניתוח מתחרים שיטתי — לא כדי לקנא, אלא כדי לנצח.

---

## שלב 1: מיפוי המתחרים

### קטגוריות מתחרים

```
Direct Competitors
└── אותו פתרון, אותו קהל, אותה קטגוריה

Indirect Competitors
└── פתרון שונה, אותו קהל, אותה בעיה

Substitute Competitors
└── "לא לקנות כלום" / לפתור אחרת לגמרי

Aspirational Competitors
└── ענקים שאתה רוצה להיות כמוהם יום אחד
```

### כלי מיפוי: Perceptual Map

```
        גבוה [ציר Y]
           |
    B      |       A (אתה)
           |
-----------+----------- [ציר X]
           |           נמוך
    C      |       D
           |
        נמוך
```

_בחר 2 ממדים רלוונטיים לענף שלך (מחיר/איכות, מסורתי/חדשני, niche/broad וכד')_

## שלב 2: Intelligence Collection

### מה לאסוף על כל מתחרה

#### Digital Footprint
```
☐ Traffic: SimilarWeb (Volume, Sources, Trends)
☐ SEO: Ahrefs/SEMrush (Keywords, DR, Backlinks)
☐ Paid: Facebook Ad Library, SpyFu
☐ Social: Followers, Engagement, Content
☐ Reviews: Google, Trustpilot, G2, App Store
☐ Jobs: LinkedIn (מה מגייסים? = מה בונים)
☐ Press: מה אמרו עליהם בחדשות?
```

#### Product Intel
```
☐ הירשם לTrial / Free Plan
☐ קרא Onboarding מיילים
☐ בדוק Pricing Page (מה Tier? מה Features?)
☐ הסתכל על Changelog / Release Notes
☐ ראה Demo / Webinar אם יש
```

#### Marketing Intel
```
☐ תיעד מודעות פעילות (Ad Library)
☐ נרשם לNewsletter שלהם
☐ עקוב אחרי Social שלהם 30 יום
☐ בדוק Content / Blog (נושאים, תדירות, תגובות)
☐ מצא Partnerships / Integrations
```

## שלב 4: Reviews Analysis — הזהב האמיתי

### G2 / Trustpilot / Google Reviews Framework

**סרוק 50+ ביקורות ותשמור:**

```
PROS (מה אוהבים?)
├── חוזר הכי הרבה: [X]
├── שני בתדירות: [Y]
└── מפתיע: [Z]

CONS (על מה מתלוננים?)
├── חוזר הכי הרבה: [X] ← כאב שאתה יכול לפתור
├── שני בתדירות: [Y]
└── Critical: [Z]

JOBS TO BE DONE (למה קנו?)
├── "רציתי ש..." [X]
├── "הייתי צריך ש..." [Y]
└── "הבעיה שלי הייתה..." [Z]
```

**זה יתן לך:**
- Messaging שמדבר בשפת הלקוח
- Positioning שמנצל חולשות מתחרים
- Features Roadmap מבוסס ביקורות אמיתיות

## שלב 5: Win/Loss Analysis

> אחד המחקרים הכי ערכותיים שרוב חברות לא עושות.

### לאחר כל Deal שנסגר (Win) ושהלך לאיבוד (Loss):

**שאל:**
1. "מה היה הגורם המכריע בהחלטה?"
2. "מי עוד הצעתם לשקול?"
3. "מה המתחרה הציע שאנחנו לא?"
4. "מה כמעט גרם לכם לבחור אחרת?"
5. "מה היה צריך להיות שונה כדי שתבחרו בנו?"

3. WHAT TO COMMUNICATE:
   "מה אנחנו צריכים להגיד שאחרים לא אומרים?"

   "מה Feature / שירות יהיה Game Changer שלנו?"
